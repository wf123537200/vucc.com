依赖

from-item

from-item 参数

label 标题

  • · 类型： String

labelCol label占grid列数,[0, 24]

  • · 类型： String | Number

wrapperCol input占grid列数,[0, 24]

  • · 类型： String | Number

isRequire 是否必填,带小红点

  • · 类型： Boolean

explainType 是否必填,带小红点

  • · 类型： String

  • · 可选值： error | success | warning | feedback

isExplainShow 是否显示提示文字

  • · 类型： Boolean

explainText 提示文字

  • · 类型： String

from 参数

isVertical 是否竖向排列

  • · 类型： Boolean

  • · 默认值： false

isValid 表单是否验证完成

  • · 类型： Boolean

  • · 默认值： false
