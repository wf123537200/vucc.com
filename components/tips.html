说明

tips组件提供了两种实例化方式,
1,可以采用常规的方法对tips进行声明,如:

2,为了方便代码中使用,tips类的弹出窗口还提供了另一种纯js的实例化机制

当tips弹窗被实例化时,将会将当前tips弹窗的vue对象引用绑定到 vm.$root.$$tips 上,以方便在任何vue对象内部对tips进行调用

参数

id 必填,用于指代整个tips实例,后续调用时通过此id调用组件实例

  • · 类型： String

type tips弹窗的类型

  • · 类型： String

  • · 默认值： info

  • · 可选值： success | confirm | error

showTime 弹窗消失的时间

  • · 类型： Number

  • · 默认值： 1200

content tips弹出框内容

  • · 类型： String
