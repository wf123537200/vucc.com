日期选择

依赖

步进输入框组件
按钮组件
输入框组件

单组件引入

import from ‘vucc/datepicker’;

参数

time 传入的日期,当isRange为false时使用

  • · 默认值： Date.now()

startTime 传入的起始日期,当isRange为true时使用

  • · 默认值： Date.now()

endTime 传入的结束日期,当isRange为true时使用

  • · 默认值： Date.now()

isRange 是否是范围类型的日期选择器

  • · 类型： Boolean

  • · 默认值： false

format 日期的输出格式,现在支持 “YYYY(-)MM(-)DD” / “”YYYY(-)MM(-)DD hh:mm:ss” 连接符号可以自选

  • · 类型： String

  • · 默认值： YYYY-MM-DD

hasFooter 是否有确定,取消两个按钮,当isRange和有时分秒的情况下不起作用

  • · 类型： Boolean

  • · 类型： false

disableFilter 禁用过滤器,返回true时将禁用改天的选择, 只有在初始化时调用,当isForceRefresh为true时,每次都会调用过滤器

  • · 类型： Function

isForceRefresh 是否强制刷新,为true时,每次都调用禁用过滤器,性能会有些许降低

  • · 类型： Boolean

  • · 类型： false

isDisabled 日期选择器是否禁用

  • · 类型： Boolean

  • · 类型： false

onOk 确定按钮回调

  • · 类型： Function

onCancel 取消按钮回调

  • · 类型： Function

onSuccess 选中按钮回调

  • · 类型： Function
