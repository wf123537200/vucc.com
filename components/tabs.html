标签页

子组件 pane

vPane组件只在vTabs组件中使用,再别处使用将只产生一个section标签来包装代码

单组件引入

import from ‘vucc/tabs’;
import from ‘vucc/pane’;

模板说明

标签页组件使用了slot分发技特性详见,主要分发3块自定义内容
分别是:头部的额外操作部分name = operate,头部部分name = header,页签内容部分:name = panes

参数

type 标签页样式类型

  • · 类型： String

  • · 默认值： default

  • · 可选值： primary|outline|error|link

size 标签页样式类型

  • · 类型： String

  • · 默认值： normal

  • · 可选值： large|small

activeIndex 当前选中的标签页

  • · 类型： Number

  • · 默认值： 0
