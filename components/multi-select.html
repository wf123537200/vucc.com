下拉多选框

下拉选择框内部直接调用了下拉列表组件,也可以直接参考下拉列表

单组件引入

import from ‘vucc/multi-select’;

参数

data 传入组件数据,用于渲染
data内部的3个可选值,展示label的优先级为,randerLi > label > value,具体使用方法参看综合示例

  • · 类型： Object

  • · 默认值：

resultList 绑定外部数据对象的结果

  • · 类型： Array

isDisabled 下拉列表是否可以选择

  • · 类型： Boolean

  • · 默认值： false

isOpened 下拉列表是否显示

  • · 类型： Boolean

  • · 默认值： false

cancelText 取消按钮的展示文字

  • · 类型： String

  • · 默认值： 取 消

okText 确定按钮的展示文字

  • · 类型： String

  • · 默认值： 确 定

onOk 确定按钮的回调函数

  • · 类型： Function

onCancel 取消按钮的回调函数

  • · 类型： Function
