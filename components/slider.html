滑动条

参数

max 拖拽的最大值

  • · 类型： String

  • · 默认值： 100

min 拖拽的最小值

  • · 类型： String

  • · 默认值： 0

text 拖拽值的附加文本

  • · 类型： String

type 当type为percent的时,将附加text = %

  • · 类型： String

value 当前滑动条的值

  • · 类型： String | Number

  • · 默认值： 100

isDisabled 拖拽的最大值

  • · 类型： Boolean

  • · 默认值： false

valueList 固定点的点阵渲染列表,如果传入,则按照固定点阵进行拖拽

  • · 类型： Array
