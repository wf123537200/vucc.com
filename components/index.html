asValue / asLabel

asValue 由于有些数据输入不规则,可能不是采用value作为key,所以这里可以传入一个字符串,指定用于渲染数据的key
asLabel 由于有些数据输入不规则,可能不是采用label作为key,所以这里可以传入一个字符串,指定用于渲染数据的key

支持组件radio, checkbox select

  • · 类型： String
