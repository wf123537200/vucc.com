下拉列表

用于给选择框和多选框等提供下拉列表,也可以单独调用

参数

data 传入组件数据,用于渲染
data内部的3个可选值,展示label的优先级为,randerLi > label > value,具体使用方法参看综合示例

  • · 类型： Object

  • · 示例：

———- 单选参数 ————–
isOpened 下拉列表是否显示

  • · 类型： Boolean

  • · 默认值： false

value 下拉列表的值

  • · 类型： String | Number

  • · 默认值： ``

onSelect 下拉列表选择函数

  • · 类型： Function

  • · 传入参数： index 当前选择的元素在列表中的位置, value 当前选择元素的值

isDisabled 下拉列表是否可以选择

  • · 类型： Boolean

  • · 默认值： false

isEditAble 下拉列表是否可编辑,如果可编辑,则展示框变成输入框,并自带过滤功能

  • · 类型： Boolean

  • · 默认值： false

filter 下拉列表选项过滤器,传入Function时,会自动传入data.optsList作为参数,方便过滤,当传入Number | String时,会自动对data.optsList做模糊过滤,

  • · 类型： Number | String | Function

———- 多选参数 ————–
resultList 绑定外部数据对象的结果

  • · 类型： Array

isDisabled 下拉列表是否可以选择

  • · 类型： Boolean

  • · 默认值： false

isOpened 下拉列表是否显示

  • · 类型： Boolean

  • · 默认值： false

cancelText 取消按钮的展示文字

  • · 类型： String

  • · 默认值： 取 消

okText 确定按钮的展示文字

  • · 类型： String

  • · 默认值： 确 定

onOk 确定按钮的回调函数

  • · 类型： Function

onCancel 取消按钮的回调函数

  • · 类型： Function
