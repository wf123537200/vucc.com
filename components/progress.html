进度条

单组件引入

import from ‘vucc/progress’;

参数

progress 组件进度百分比,取值在[0, 100]

  • · 类型： Number

type 组件的的类型

  • · 类型： String

  • · 默认值： normal

  • · 可选值： normal | success | error | warn

size 组件的的类型

  • · 类型： String

  • · 默认值： normal

  • · 可选值： normal | large

title 是否有title,决定是否显示title栏

  • · 类型： String

info 是否有info提示,决定是否显示title栏

  • · 类型： String

explain 说明文字,决定是否显示explain部分

  • · 类型： String

isShowText 是否显示progress

  • · 类型： Boolean

  • · 默认值： false

isActive 是否展示动态类型的progress

  • · 类型： Boolean

  • · 默认值： false
