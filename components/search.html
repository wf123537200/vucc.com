参数

  • · 类型： String

size 按钮的大小

  • · 类型： String

  • · 默认值： normal

  • · 可选值： large | small

placeholder placeholder文案

  • · 类型： String

onSearch 点击回车或者搜索按钮回调函数

  • · 类型： Function
