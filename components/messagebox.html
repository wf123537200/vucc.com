messageBox类弹窗

和tips相比,messagebox类弹窗只有一种调用方式,就是通过声明式进行实例化

参数

id 必填,用于指代整个弹窗实例,后续调用时通过此id调用组件实例

  • · 类型： String

  • · 默认值： ``

type 弹窗的类型

  • · 类型： String

  • · 默认值： info

  • · 可选值： success | confirm | error

title 弹出框主标题

  • · 类型： String

explain 弹出框说明文字

  • · 类型： String

cancelText 取消按钮的展示文字

  • · 类型： String

  • · 默认值： 取 消

okText 确定按钮的展示文字

  • · 类型： String

  • · 默认值： 确 定

onOk 确定按钮的回调函数

  • · 类型： Function

onCancel 取消按钮的回调函数

  • · 类型： Function
